# 🛡️ מוקד דוברות — מערכת ניטור תקשורת
### משטרת ישראל | אגף דוברות

מערכת ניטור מקצועית לעקיבה אחרי ידיעות חדשות בזמן אמת, שידורי חיים, ופס רץ של כותרות.

---

## 🚀 הוראות הפעלה ב-GitHub Pages — שלב אחר שלב

### שלב 1 — פתח את GitHub
1. כנס לאתר: **https://github.com**
2. היכנס עם שם המשתמש והסיסמה שלך

---

### שלב 2 — צור Repository חדש
1. לחץ על כפתור **"New"** (בצד שמאל למעלה, ירוק)
2. מלא את הפרטים:
   - **Repository name:** `police-monitor` (או כל שם אחר)
   - **Description:** מוקד דוברות — מערכת ניטור תקשורת
   - בחר **Public** (חינם, נדרש ל-GitHub Pages)
   - ✅ סמן **"Add a README file"**
3. לחץ **"Create repository"** (ירוק, בתחתית)

---

### שלב 3 — העלה את קובץ ה-HTML
1. בעמוד ה-Repository שנפתח, לחץ **"Add file"** ← **"Upload files"**
2. גרור את הקובץ `index.html` לחלון, או לחץ **"choose your files"**
3. בתחתית הדף, בשדה **"Commit changes"**, כתוב: `הוספת מערכת ניטור`
4. לחץ **"Commit changes"** (ירוק)

---

### שלב 4 — הפעל GitHub Pages
1. לחץ על **"Settings"** (בשורת הכפתורים בחלק העליון של ה-Repository)
2. בתפריט השמאלי, גלול למטה ולחץ **"Pages"**
3. תחת **"Branch"**, בחר:
   - Branch: **main**
   - Folder: **/ (root)**
4. לחץ **"Save"**

---

### שלב 5 — קבל את הקישור שלך
1. המתן **2-3 דקות**
2. רענן את דף Settings → Pages
3. תראה הודעה ירוקה: **"Your site is published at..."**
4. הקישור יהיה בפורמט: `https://[שם-המשתמש].github.io/police-monitor/`

**זהו! המערכת פועלת 🎉**
